This website is about "Cascading Style Sheets & HTML"

This page demonstrates external CSS styling. The styles are defined in a separate CSS file and linked to this HTML document.

Method Description Example
In-line Styles embedded directly in HTML tags <body style ="background-color:#a0dfd9"></body>
Internal Styles defined in the head section 

<style>
body {background-color:#a0dfd9}
h1 {
    color:blue;
    text-align:left;
    font-size:16px;
    background: #ffffff;
    width:600px
}
</style>
External Styles in separate CSS file <link rel="stylesheet" href="styles.css">

Benefits of CSS

CSS provides consistency, reduces workload, and aids maintenance across websites.
