Database Setup:

Step 1.)

Run the following SQL commands if you want to create tables manually:

CREATE DATABASE petshop CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
USE petshop;

Step 2.)

Enter this URL to your browser (e.g. Google Chrome, Microsoft Edge)

http://localhost/Example_petfood*/db.php

This activates the database of the website.
